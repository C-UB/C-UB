FAQ

使用该系统需要有哪些条件？

1. 基础条件，可以上网的计算机
2. 安装 metamask 插件，作为电子钱包，管理账户信息
3. 按照metamask 提示来操作，注册账户，登录
4. 牢记助记词，牢记助记词，牢记助记词！丢失后无法找回！

该系统有哪些功能

1. 资源发布与分享，资源市场浏览。
2. 资源版权保护，下载资源的花费直接递交作者账户
3. 记录学信大数据，开放查询，用人单位一键验证，省时省力。
4. 课下自主学习社区，提供练习和讨论，期末测评表现。

系统和区块链是怎样的关系

1. 系统采用的是spring boot + mybatis + 前端界面来进行功能接口的展示
2. 区块链使用以太坊，智能合约使用solidity 语言开发，目前部署在 Rinkeby 测试网络
3. 系统与区块链使用web3js 进行交互，对区块链的各种请求均不通过服务器，服务器仅仅返回静态页面
4. 该系统被攻击后用户账户不会遭受任何损失，仅仅在短时间内无法通过网页访问区块链（使用命令行依旧可以直接访问）
